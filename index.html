Welcome to ResponsivePDF pages.

PDF was made to be static and to ensure that it's visual presentation will be the same on each device and platform. However PDF format can contain interactive elements, which can make document content to be adaptable to locale, output device, viewing conditions.

PDF Layers

PDF Layers or Optional Content allows showing and hiding the content based on user selection or viewing conditions. It's main purpose is to turn on-off different content on CAD based documents, but it can do much more. Localised content, Maps are good examples.

How to try

Grab some samples responsive pdf files from the repository, open it in your pdf viewer on different devices, play with different views, zooms.

Adobe Acrobat plugin

The plugin can help you with creating the responsive pdf. See the wiki page for more information (https://github.com/responsivepdf/responsivepdf/wiki).
